An HTML form is a section of a document which contains controls such as text fields,
password fields, checkboxes, radio buttons, submit button, menus etc.An HTML form for
facilitates the user to enter data that is to be sent to the server processing such as
name, email address, password, phone number, etc.HTML forms are required if you want to collect some data
