CSS Attributes

CSS id Selector

id selector uses the id attribute of an HTML element to select a specific element.
To select an element with a specific id, write a hash (#) character, followed by the id of the element.

CSS Class Selector

The class selector selects HTML elements with a specific class attribute.
To select elements with a specific class, write a period (.) character, followed by the class name.

CSS Universal Selector

Every element on the page will get affected . write * for universal selector


CSS Grouping Selector

grouping selector selects all the HTML elements with the same style definitions.

Three Ways to Insert CSS

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS: unique style for a single element.

Internal CSS: single HTML page has a unique style.

External CSS: you can change the look of an entire website by changing just one file! Each HTML page must include a reference to the external style sheet file inside the element, inside the head section.
